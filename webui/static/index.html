Runs K simulations on the selected geometry. Iter-0 is a baseline uniform-saturation run. Each subsequent iteration uses integral accumulation: a small correction Δs = −m × (T̃ − 0.5) is added to the prior saturation map using the iter-0 normalization as a fixed reference. This prevents residual re-amplification (the "glass ceiling" of proportional mode) and avoids the hot/cold field inversion that drives oscillation.

Empirical optimum (circle, 4 bpp): m = 0.7, n = 12, decay = 1.0 → σT = 4.43°C at iter-9 (77% reduction; 37% below the best single proportional step of 7.05°C). An early-stop triggers automatically when σT rises for 2 consecutive iters — the best map is used.

Geometry guidance: convex shapes (circle, hexagon) respond best. Diamond and re-entrant geometries are better handled by turntable rotation. Near-uniform shapes (σT < 6°C at iter-0) should not use FGM.
